# Servicio Social

Este repositorio contiene el trabajo realizado durante mi servicio social, incluyendo documentación y código desarrollado para  aplicaciones con ESP32.

## Contenido del Repositorio

- **Apuntes**: Documentación y notas relacionadas con el servicio social.
- **Codigos**: Scripts y programas desarrollados, principalmente enfocados en el uso del ESP32.

## Tecnologías Utilizadas

- **ESP32**: Microcontrolador principal utilizado en los proyectos.
- **Arduino IDE / PlatformIO**: Entornos de desarrollo para la programación del ESP32.
- **Lenguaje C/C++**: Desarrollo del código fuente.

## Propósito

El objetivo de este repositorio es documentar y compartir el conocimiento adquirido durante mi servicio social, proporcionando ejemplos de código y recursos útiles para futuros desarrollos.
